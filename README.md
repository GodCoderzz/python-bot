# 🚀 Run Locally

  

Clone the project

  

```bash

git clone https://github.com/GodCoderzz/python-bot.git

```

  

Go to the project directory

  

```bash

cd my-project

```

  

Install dependencies

  

```bash

pip install -r requirements.txt

```

  

Start the bot

  

```bash

python .\automating.py

```
